# drizzleDumper
  
          
## 简介
###*drizzleDumper*是一款基于内存搜索的Android脱壳工具。
######*drizzleDumper* is a memory-search-based Android unpack tool.
  
  
  
##使用方法
	./drizzleDumper package_name wait_times(s)
